Tema: Objetos

Preguntas de interes:

¿Qué es un objeto en JS?

Los objetos son variables, pero puecen contener muchos valores

Diferencia entre un objeto y una variable

Los objetos pueden contener muchos valores, mientas que las variables no

¿Cómo se crea un objeto en JS?

Se crea como una variable, pero se le abren corchetes para asignarle los valores

Ejemplo de como crear un objeto en Js:

< script > const persona = { nombre: "Lucio", Apellido: "Rasgado", Edad: 25, colorojos: "Negros" }; document.getElementById("datos").innerHTML = persona.nombre + " " + persona.Apellido + " tiene " + persona.Edad + " años de edad y tiene los ojos de color " + persona.colorojos; < /script >
