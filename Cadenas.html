Tema: Cadenas

Preguntas de interes:

¿Qué es una cadena en Js?

Una cadena de JavaScript es cero o más caracteres escritos entre comillas.

¿Para qué sirven las cadenas en Js?

Las cadenas de JavaScript son para almacenar y manipular texto.

¿Cómo se forma una cadena en Js?

Puedes utilizar comillas simples o dobles:
let nombre1 = "Carlos" (Comillas dobles)
let nombre2 = 'Maria' (Comillas simples)

Ejemplo de una cadena:

< script > let nombre1 = "Jose"; let nombre2 = 'Carlos'; document.getElementById("demo").innerHTML = nombre1 + " " + nombre2; < /script >
