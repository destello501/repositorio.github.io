Tema: Bucles

Preguntas de interes:

¿Qué es un bucle?

Un bucle es un bloque de codigo que se puede ejecutar varias veces

¿Para que sirve un bucle?

Los bucles son útiles si desea ejecutar el mismo código una y otra vez, cada vez con un valor diferentes A menudo, este es el caso cuando se trabaja con matrices

¿Qué tipos de bucles hay?

JavaScript admite diferentes tipos de bucles:

  • for- recorre un bloque de código varias veces
  • for/in- recorre las propiedades de un objeto
  • for/of- recorre los valores de un objeto iterable
  • while- recorre un bloque de código mientras una condición específica es verdadera
  • do/while- también recorre un bloque de código mientras una condición específica es verdadera

Declaración (for):

< p id="demo" > < /p > < script > var i = 5; for (var i = 0; i < 10; i++) { } document.getElementById("demo").innerHTML = i; < /script >
